== 请务必注意 ==

「盐」建议在每个游戏中都保持不同，防止被别人猜测出你的选项

「盐」请不要设置为你的常用密码，因为在公布出手选项时任何人都可以看到你的盐

请记住你参与的游戏的ID、盐以及出手选项，否则将会导致你无法公开自己的选项

================

# 剪刀石头布

使用密钥或者密钥文件登录后，你可以进行如下操作

## 新建游戏

设置好游戏的金额（不建议设置太多，玩玩就好）、盐和出手选项后即可提交

**请务必记住盐和出手选项**

提交到链的参数为 MD5(choice + '[email protected][email protected]' + salt)

新建完成后，可以通过刷新「游戏大厅」或者「我的游戏」来查看刚刚的游戏 ID

## 加入游戏

如果你看到游戏的状态为「等待加入」的，你可以加入这个游戏

设置好游戏的盐和出手选项后即可提交

**如果该游戏被新建的玩家设置了一个投注金额，在加入的同时你也会发送相等的金额到合约中**

**请务必记住盐和出手选项**

提交到链的参数为 id 和 MD5(choice + '[email protected][email protected]' + salt)

## 公开选项

当目前的游戏有两人参加时（游戏状态为「等待公开」），游戏的两位玩家可以选择公开自己的出手选项

设置好刚刚参加游戏时的盐和出手选项后即可提交

提交到链的参数为 id 和 出手选项 和 盐

合约会判断提交的选项和盐是否正确，如果正确则会保存下来

如果两位玩家均提交了正确的选项和盐，系统会开始判定游戏的胜负

胜利的一方可以获得全部的奖金（即双倍于自己的投注金额）；如果平手，则会退还双方的投注

## 强行结束

1. 没有人迎战，即只有一个玩家参与游戏，该玩家可以提出强行结束当前游戏并退回自己的投注
2. 在游戏玩家公开了自己的出手选项的一天之后，如果另外一名玩家并没有公开自己的出手选项，未公开出手选项的玩家将视作认输，但仍可以公开自己的选项。
此时任何人都可以提出结束这个游戏，则公开了自己的出售选项的玩家获得全部奖金
